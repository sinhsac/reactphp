# reactphp

## Cài đặt composer trước tiên 

[https://getcomposer.org/](https://getcomposer.org/)

## Chạy lệnh sau trong thư mục dự án
`composer install`

## Sửa config.php để thay đổi port của server

## chạy lệnh sau để start server
`php main.php`

## chạy server dạng daemon 
`nohup php main.php &`
